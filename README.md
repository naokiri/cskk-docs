# 概要

ユーザ向けCSKKおよびfcitx5-cskkの説明書

[honkit](https://github.com/honkit/honkit) を使用して静的なページを生成している。 

# 開発手順

    # nvm use
    # npm install
    # npm run dev

# デプロイ手順
(そのうちgithub actionsで自動化する)

    # npm run build

_book ディレクトリに静的ページができる。

pdfやebook等の生成方法については[honkit](https://github.com/honkit/honkit)参照のこと
